## 📌 Обзор

AgInTi сейчас находится на этапе инициализации/подготовки каркаса. На момент создания этого черновика README репозиторий в основном содержит:

- Git-метаданные (`.git/`)
- Каталог `i18n/`, подготовленный для многоязычных файлов README
