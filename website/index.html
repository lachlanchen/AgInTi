LRE Pipeline

Deterministic stage chain with fallback behavior and latest-artifact promotion.

1) Profile Identity anchors, source validation, and evidence quality checks.
2) Books Growth-focused reading plan derived from profile strengths and gaps.
3) Investments Thesis candidates and risk framing aligned with current objectives.
4) Ideas Research directions, milestone draft, and self-heal checkpoints.

Course Agent Pipeline

Human-like action loop for online course completion and workflow maintenance.

A) Bootstrap Open login session with browser automation and controlled checkpoints.
B) Build Plan Create structured tool plan from objective and prior issue logs.
C) Autodev Generate/fix tools inside target root and reuse runtime primitives.
D) Iterate Use result artifacts to refine prompt logic and reduce failure loops.
